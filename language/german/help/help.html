Hilfe: wgSimpleAcc

Beschreibung

Das XOOPS Modul wgSimpleAcc dient zur Erfassung einer Einnahmen-/Ausgabenrechnung für Vereine und kleine Unternehmen.

Installieren/Deinstallieren

Keine speziellen Massnahmen erforderlich, bitte führen Sie den Standardinstallationsprozess durch: Entpacken die den Ordner wgteams in das Verzeichnis '../modules'. Installation über Administration -> System Module -> Module.

Wenn Sie zustätzliche Informationen benötigen, lesen Sie die Installationshinweise unter XOOPS Operations Manual

Funktionen

Wichtige Funktionen:

  • Verwendung von Zuordnungen, Vermögenswerten und Konten
  • Detailierte Rechteverwaltung (Bestätigen, Senden, Ansehen)
  • Verwendung eines Kundenmanagements (Kreditoren und Debitoren)
  • Verwendung von Ausgabevorlagen
  • Visualisierung mit Chartjs
