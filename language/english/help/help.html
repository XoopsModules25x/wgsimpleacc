Help: wgSimpleAcc

Description

The wgSimpleAcc is a XOOPS module to record an income / expense account for clubs or small businesses.

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgsimpleacc folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Important features:

  • Use of assignments, assets and accounts
  • Detailed rights management (confirm, send, view)
  • Use of client management (creditor and deptors)
  • Use of your own output templates
  • Visualization with Chartjs
